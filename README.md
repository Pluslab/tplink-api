# tplink-api

Go library for TP-Link HS105 smart plug

## Description

Go library for TP-Link HS105 smart plug.

The SmartHome protocol runs on TCP port 9999.

## Requirements

| 言語/FW        | Version |
| :------------- | ------: |
| go             |  1.11.5 |

## Installing

```sh
go get github.com/Pluslab/tplink-api
```

## Usage

```go
import "github.com/Pluslab/tplink-api"
```
